# CampusOS v0.4.5-dev 进度说明

> 日期：2026-07-01
> 状态：已完成
> 范围：个人主页 Space 后端基础

## 1. 本次目标

进入 v4 计划中的“个人主页/博客园插件”任务，先完成可独立验收的后端基础：公开个人主页入口、用户主页配置数据模型、PostgreSQL/内存仓储、登录用户自助保存接口和帮助文档。

本次不把事件驱动内容同步、前端主页渲染、风格包导入导出混入同一提交，避免范围过大。

## 2. 完成内容

| 项目 | 结果 |
| --- | --- |
| 个人主页模型 | 新增 `Space`、`Owner`、`PublicSpace` 和 `UpsertSpaceRequest` |
| 公开主页入口 | 新增 `GET /api/v1/space/:user_id` 和 `GET /api/v1/u/:username` |
| 用户自助管理 | 新增 `GET /api/v1/spaces/me` 和 `PUT /api/v1/spaces/me` |
| 默认主页 | 用户未保存配置时自动返回默认主页，`is_default=true` |
| 可见性控制 | 支持 `public`、`unlisted`、`private`，公开接口只返回 `public` |
| 同步筛选字段 | 预留 `sync_enabled`、`sync_categories`、`sync_tags` |
| PostgreSQL 持久化 | 新增 `user_spaces` 表、唯一索引和同步筛选 GIN 索引 |
| 内存模式 | 新增 Memory Repository，保持数据库不可用时服务仍可启动 |
| 测试覆盖 | 覆盖默认主页生成、配置保存、私有主页公开访问拒绝、JWT 用户绑定 |
| 帮助文档 | 新增个人主页 Space 使用说明 |
