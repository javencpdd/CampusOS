# CampusOS v0.4.19-dev 进度说明

> 日期：2026-07-02
> 状态：已完成
> 范围：修复管理后台新建版块返回 400 的问题

## 1. 本次目标

修复在 `http://localhost:3001/categories` 点击“新建版块”后创建失败的问题。

用户侧现象：

```text
POST /api/v1/categories
HTTP 400
```

## 2. 原因分析

后端 `CreateCategoryRequest` 旧逻辑要求 `slug` 必填：

```go
Slug string `json:"slug" binding:"required,min=1,max=64"`
```

但管理后台“新建版块”表单只提交：

```text
name, description, icon, color, sort_order
```

没有提交 `slug`。因此请求在 Gin JSON 绑定阶段被拦截，业务逻辑没有执行，直接返回 `400`。
