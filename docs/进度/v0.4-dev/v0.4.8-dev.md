# CampusOS v0.4.8-dev 进度说明

> 日期：2026-07-01
> 状态：已完成
> 范围：个人主页风格包导入预览

## 1. 本次目标

继续推进 v4 计划中的“主页风格导入导出插件”。在 v0.4.7 已完成风格包 manifest 与安全校验后，本次增加导入预览能力，让用户上传风格包后可以先看到安全校验结果和规范化后的可渲染预览模型。

本次仍不保存、不安装、不应用风格包，避免在预览阶段引入持久化和回滚复杂度。

## 2. 完成内容

| 项目 | 结果 |
| --- | --- |
| manifest 规范化 | 新增 `NormalizeStyleManifest`，统一 trim、默认 layout 和数组/map 规范化 |
| 预览模型 | 新增 `StylePreview`，包含校验结果、当前用户、当前主页、布局、组件、token 和资源声明 |
| 预览构建 | 新增 `BuildStylePreview`，校验失败时不暴露可渲染 manifest |
| Service 接入 | 新增 `PreviewStylePackage`，绑定当前登录用户主页上下文 |
| API 接口 | 新增 `POST /api/v1/spaces/me/styles/preview` |
| 测试覆盖 | 覆盖有效预览、无效预览不返回 manifest、handler 响应 |
| 帮助文档 | 更新风格包说明，补充导入预览 API |
