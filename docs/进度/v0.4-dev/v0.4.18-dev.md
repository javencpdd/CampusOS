## 1. 本次目标

支持通过系统配置文件选择账号密码保存方式：

| 模式 | 行为 |
| --- | --- |
| 默认哈希模式 | 注册时把密码保存为 bcrypt，登录时 bcrypt 校验 |
| 本地明文调试模式 | 注册时把密码明文保存到数据库，登录时直接按明文比对 |

同时在 `scripts/` 下提供 bcrypt 哈希生成脚本，方便后续把明文密码转换为哈希后再保存。

## 5. 使用说明

默认安全模式：

```env
AUTH_PASSWORD_HASH_ENABLED=true
```

本地明文调试模式：

```env
AUTH_PASSWORD_HASH_ENABLED=false
```

修改 `.env` 后需要重启后端。

生成 bcrypt：

```bash
scripts/hash-password.sh '明文密码'
```

或交互输入：

```bash
scripts/hash-password.sh
```

注意：明文模式只用于本地临时调试。切回哈希模式前，需要把明文 `accounts.credential` 转换为 bcrypt 哈希。
