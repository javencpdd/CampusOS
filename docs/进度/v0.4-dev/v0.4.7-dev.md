# CampusOS v0.4.7-dev 进度说明

> 日期：2026-07-01
> 状态：已完成
> 范围：个人主页风格包 manifest 与安全校验

## 1. 本次目标

进入 v4 计划中的“主页风格导入导出插件”任务，先完成 P0 的风格包格式基础：定义结构化 manifest、受控组件、样式 token、资源声明和安全校验规则。

本次只做校验和格式约束，不保存、不导入、不应用风格包，为后续导入预览、导出、一键切换和回滚提供共同基础。

## 2. 完成内容

| 项目 | 结果 |
| --- | --- |
| 风格包格式 | 新增 `StylePackage`、`StyleManifest`、`StyleComponent`、`StyleAsset` |
| schema 版本 | 当前固定为 `space-style.v1` |
| 受控布局 | 支持 `blog`、`grid`、`timeline`、`magazine` |
| 受控组件 | 支持 profile header、content list、tag cloud 等白名单组件 |
| token 校验 | 只允许 `color.`、`font.`、`space.`、`radius.`、`shadow.` 前缀 |
| 资源校验 | 只允许包内相对路径和 png/jpg/jpeg/webp 图片资源 |
| 安全校验 | 拒绝脚本组件、事件 props、HTML/script 语义、外链资源和路径穿越 |
| 校验 API | 新增 `POST /api/v1/spaces/me/styles/validate` |
| 测试覆盖 | 覆盖合法 manifest、脚本组件、外链资源、路径穿越、危险 token 和 handler |
| 帮助文档 | 新增个人主页风格包说明 |
