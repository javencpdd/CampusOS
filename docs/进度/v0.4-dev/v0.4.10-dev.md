# CampusOS v0.4.10-dev 进度说明

> 日期：2026-07-01
> 状态：已完成
> 范围：个人主页风格包导出

## 1. 本次目标

继续推进 v4 计划中的“主页风格导入导出插件”。在 v0.4.7 完成风格包 manifest 与安全校验、v0.4.8 完成导入预览后，本次完成 P0 的“从当前个人主页配置导出风格包”能力。

本次只导出标准风格包 JSON，不保存、不安装、不应用风格包，也不打包图片资源或用户内容数据，为后续“一键应用”和“回滚快照”继续收敛数据边界。

## 2. 完成内容

| 项目 | 结果 |
| --- | --- |
| 导出请求 | 新增 `StyleExportRequest`，支持指定导出名称、版本和描述 |
| 导出响应 | 新增 `StyleExportResult`，返回风格包、建议文件名和校验结果 |
| manifest 生成 | 从当前用户主页的 `layout`、`theme` 和同步标签状态生成受控组件与 token |
| 名称规范化 | 导出名称会转换为小写短横线格式，未传时由用户名生成 |
| 版本校验 | 未传版本默认 `0.1.0`，非法语义版本会拒绝导出 |
| 布局兜底 | 不支持的主页布局导出时回退为 `blog`，保证导出包可校验 |
| 主题 token | 支持 `default`、`ink`、`dark`、`warm` 到受控 token 的转换 |
| API 接口 | 新增 `POST /api/v1/spaces/me/styles/export` |
| 测试覆盖 | 覆盖默认导出、自定义非法版本拒绝和 JWT handler 响应 |
| 帮助文档 | 更新个人主页 Space 与风格包说明文档 |
