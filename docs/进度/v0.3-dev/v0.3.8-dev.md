# CampusOS v0.3.8-dev 进度说明

> 日期：2026-06-30
> 状态：已完成

## 1. 本次目标

完成 v0.3-dev 中的 Wasm 事件 payload ABI，让 Runtime 可以把 `plugin.EventMessage` 以 JSON bytes 的形式传入 Wasm 插件，同时保留 `hello-wasm` 当前使用的无参数兼容 ABI。

## 2. 完成内容

| 项目 | 结果 |
| --- | --- |
| ABI 识别 | `handle_event()` 继续作为兼容 ABI；`handle_event(i32 ptr, i32 len)` 作为 payload ABI |
| Payload 编码 | `SendEvent` 将 `EventMessage` 序列化为 JSON bytes |
| Memory 写入 | payload ABI 要求插件导出 `memory`，Runtime 写入 guest memory 后传入 `ptr/len` |
| 写入位置 | 优先使用导出的 `__heap_base` 作为 payload offset；未导出时回退到 `0` |
| 边界保护 | 增加 payload 过大、缺少导出内存、不支持函数签名等错误路径 |
| 兼容性 | 无参数 `handle_event()` 的旧测试和 `hello-wasm` 示例保持可用 |
| 文档同步 | 更新 `hello-wasm` README，说明当前 Runtime 支持两种事件 ABI |
