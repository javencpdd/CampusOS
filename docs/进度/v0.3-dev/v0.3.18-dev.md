## 4. 建议的 Windows 实机验证命令

在 Windows 11 或 Windows 10 + WSL2/Docker Desktop 环境中执行：

```powershell
Copy-Item .env.example .env
.\scripts\docker-up.ps1
.\scripts\migrate.ps1 up
.\scripts\migrate.ps1 status
go test ./...
```

如果要完整启动本地服务：

```powershell
.\scripts\dev.ps1
```
