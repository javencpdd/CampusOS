# CampusOS v0.3.12-dev 进度说明

> 日期：2026-06-30
> 状态：已完成

## 1. 本次目标

完成 v0.3-dev 剩余任务中的 Go SDK 雏形，让插件开发者可以复用事件、manifest 和 Host API 请求封装，而不是直接手写底层 JSON/HTTP 细节。

## 2. 完成内容

| 项目 | 结果 |
| --- | --- |
| SDK 目录 | 新增 `sdk/go` |
| 基础类型 | 新增 `Event`、`Manifest`、权限、事件订阅、存储配置类型 |
| Host API Client | 新增 `HostClient`，默认调用 `http://127.0.0.1:18080/api/host/{Method}` |
| 插件身份 | Client 自动设置 `X-CampusOS-Plugin` header |
| 配置读取 | 封装 `GetConfig` 和 `GetAllConfig` |
| 存储访问 | 封装 `StorageGet`、`StorageSet`、`StorageDelete` |
| SDK 文档 | 新增 `sdk/go/README.md` 说明当前能力和基本用法 |
| 单元测试 | 覆盖插件身份 header、配置读取、storage 命名空间和 HTTP 错误响应 |
