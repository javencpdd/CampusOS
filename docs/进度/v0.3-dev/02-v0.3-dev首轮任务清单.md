# CampusOS v0.3-dev 首轮任务清单

> 日期：2026-06-28
> 范围：v0.3-dev 第一轮开发任务拆解
> 目标：先做插件运行安全闭环，再做 SDK/CLI 和后台体验

## 1. P0 任务

| 任务 | 交付物 | 验收标准 |
| --- | --- | --- |
| Runtime 接口复核 | `internal/plugin/runtime.go` 维持稳定或小幅补充 | gRPC Runtime 不回归，Wasm Runtime 能实现同一接口 |
| Wasm Runtime 骨架 | `internal/plugin/wasm` | 可注册 `runtime: wasm`，支持 Start/Stop/HealthCheck |
| Wasm 事件调用 | Wasm `SendEvent` 最小实现 | hello-wasm 能收到 `thread.created` 等事件 |
| 调用超时 | Runtime 调用 deadline | 插件卡住不会阻塞主服务 |
| 错误隔离 | trap/panic/error 处理 | 插件错误只影响插件状态，不影响主服务 |
| Host API 权限检查 | 权限检查器 | 未声明权限的插件不能调用敏感 API |
| 插件日志落库 | plugin_logs 写入路径 | 启动、事件、权限拒绝、错误均可记录 |
| hello-wasm 示例 | `examples/plugins/hello-wasm` | 可加载、启停、接收事件、写日志 |

## 2. P1 任务

| 任务 | 交付物 | 验收标准 |
| --- | --- | --- |
| Host API 契约文档 | `docs/进度/v0.3-dev` 文档更新 | API 名称、输入、输出、错误清晰 |
| 插件配置读取 | `GetConfig` | 插件能读取自身配置 |
| 插件 SQLite 存储设计 | 存储设计文档或最小实现 | 明确路径、隔离、备份策略 |
| 管理后台插件日志入口 | admin 页面增强 | 管理员能看到插件调用日志 |
| Windows 脚本实机验证 | 验证记录 | `.ps1` 在 Windows + Docker Desktop 上跑通 |

## 3. P2 任务

| 任务 | 交付物 | 验收标准 |
| --- | --- | --- |
| Go SDK 雏形 | `sdk/go` 或后续目录 | 示例插件不直接依赖底层 JSON 细节 |
| CLI 雏形 | `cmd/campusosctl` | 支持 `plugin init` / `plugin inspect` 的最小版本 |
| 插件打包规范 | manifest + wasm + assets 规范 | 示例插件可以按固定结构打包 |
| Windows CI | GitHub Actions matrix | 至少跑 `go test ./...` |

## 4. 建议执行顺序

1. 写 Wasm Runtime 骨架，不接 Host API。
2. 接入 Manager 注册和插件生命周期。
3. 做 hello-wasm 最小示例。
4. 接事件投递。
5. 加超时和错误隔离。
6. 加 Host API 权限检查。
7. 加 plugin_logs 写入。
8. 补测试和文档。
9. 再进入 SDK/CLI 和后台页面。

## 5. 每个任务的最低提交要求

每个实现型任务提交前应满足：

```bash
go test ./...
```

如果改动前端：

```bash
cd web && pnpm build
cd admin && pnpm build
```

如果改动 migration：

```bash
make migrate-up
make migrate-status
```

如果改动 workflow：

```bash
python3 - <<'PY'
from pathlib import Path
import yaml
for p in Path('.github/workflows').glob('*.yml'):
    yaml.safe_load(p.read_text())
    print(p, 'ok')
PY
```

## 6. 本轮不做

| 不做项 | 原因 |
| --- | --- |
| 插件市场 | 依赖 Runtime、权限、日志稳定 |
| 多语言 SDK | 先完成 Go SDK 和最小 wasm 规范 |
| 全量生产部署自动化 | 当前 CD 已能上传包，运行环境细节需要服务器侧确认 |
| 大规模 UI 重构 | 会稀释 Wasm 安全隔离主目标 |
