# CampusOS v0.3.1-dev 进度说明

> 日期：2026-06-28
> 状态：已完成

## 1. 本次目标

完成 v0.3-dev 首轮 P0 中的 Wasm Runtime 骨架，让插件系统具备可注册的 `wasm` Runtime 生命周期入口。

本次不实现真实 Wasm 执行引擎，`SendEvent` 暂时返回明确的未实现错误。真实执行会在后续 wazero 集成任务中完成。

## 2. 完成内容

| 项目 | 结果 |
| --- | --- |
| Wasm Runtime 包 | 新增 `internal/plugin/wasm` |
| Runtime 生命周期 | 实现 `Start`、`Stop`、`HealthCheck`、`IsRunning`、`Type` |
| wasm 模块定位 | 默认读取 `plugin.wasm`，支持 `manifest.config.module` 指定相对或绝对路径 |
| 事件派发边界 | `SendEvent` 对运行中的 wasm 插件返回 `ErrEventDispatchNotImplemented` |
| 服务注册 | `internal/server/server.go` 注册 `wasm` Runtime |
| 单元测试 | 覆盖生命周期、配置模块路径、模块缺失错误 |
