## 本次更新内容

### 新增文件

| 文件 | 说明 |
|:---|:---|
| `internal/plugin/manifest.go` | Manifest 结构定义、YAML 解析、权限验证 |
| `internal/plugin/runtime.go` | Runtime 接口、Plugin/EventMessage/PluginResponse 定义 |
| `internal/plugin/manager.go` | Plugin Manager 核心（注册表 + 生命周期 + 事件分发 + 批量安装） |
| `internal/plugin/grpc/runtime.go` | gRPC 运行时（进程管理 + 优雅关闭 + 健康检查 + 自动重启） |
| `internal/plugin/hostapi/hostapi.go` | Host API（Identity/Data/Event 接口） |
| `internal/plugin/handler.go` | 插件管理 HTTP 处理器 |

### 修改文件

| 文件 | 变更说明 |
|:---|:---|
| `internal/server/server.go` | 集成 Plugin Manager + gRPC Runtime，事件自动分发到插件，新增插件管理路由 |
| `go.mod` | 新增 `gopkg.in/yaml.v3` 依赖 |

### 新增 API 端点（5 个，总计 33 个）

| 方法 | 端点 | 说明 | 认证 |
|:---|:---|:---|:---|
| `GET` | `/api/v1/plugins` | 插件列表 | ✅ admin |
| `GET` | `/api/v1/plugins/:name` | 插件详情 | ✅ admin |
| `POST` | `/api/v1/plugins/:name/enable` | 启用插件 | ✅ admin |
| `POST` | `/api/v1/plugins/:name/disable` | 禁用插件 | ✅ admin |
| `DELETE` | `/api/v1/plugins/:name` | 卸载插件 | ✅ admin |

### 核心能力

- **Plugin Manager**：从 `plugin.yaml` 解析 Manifest，管理插件生命周期（installed → enabled → running → stopped）
- **gRPC Runtime**：进程管理（启动/停止）、SIGTERM 优雅关闭、10s 健康检查、崩溃自动重启
- **Host API**：`Identity.GetUser()`、`Data.QueryThreads()`、`Event.Publish()`
- **事件分发**：EventBus 事件自动异步分发到所有订阅的运行中插件
