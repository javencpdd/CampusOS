# CampusOS v0.1.8-dev 发布说明

> **版本**：v0.1.8-dev
> **发布日期**：2024-06-24
> **上一版本**：v0.1.7-dev
> **状态**：已完成

---

## 版本概述

v0.1.8-dev 完善了所有 Service 的事件发布能力，并新增了**事件历史查询 API**，使"万物皆事件"架构全面落地。现在用户注册、帖子创建/更新/删除、回复创建、版块创建都会自动广播事件，同时可通过 API 查询事件历史。

---

### 新增文件

| 文件 | 说明 |

### 修改文件

| 文件 | 变更说明 |
|:---|:---|
| `internal/community/service/thread_service.go` | 注入 EventBus，发布 thread.created/updated/deleted 事件 |
| `internal/community/service/category_service.go` | 注入 EventBus，发布 category.created 事件 |
| `internal/community/service/post_service.go` | 注入 EventBus，发布 post.created 事件 |
| `pkg/eventbus/memory_bus.go` | 新增事件历史记录（最近 1000 条）+ GetHistory() 方法 |
| `internal/server/server.go` | 传递 bus 到所有 Service，注册事件历史 API 路由 |

| `thread.deleted` | ThreadService.DeleteThread() | campusos.community |
