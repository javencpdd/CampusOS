> **修订日期**：2024-06-25

## 修订背景

v1 项目计划在项目初期制定，经过 9 个迭代版本（v0.1.0 ~ v0.1.8）的实际开发后，发现以下需要调整的地方：

### v1 → v2 核心变更

| 维度 | v1 计划 | 实际情况 | v2 调整 |
|:---|:---|:---|:---|
| **v0.1 范围** | 过大（32 API + gRPC 插件 + RBAC） | 完成约 60%（19 API） | 拆分为 v0.1（基础）+ v0.2（进阶） |
| **RBAC 权限** | 计划 v0.1 | 未实现，仅"作者权限" | 移到 v0.2 |
| **gRPC 插件** | 计划 v0.1 | 未实现 | 移到 v0.2 |
| **前端** | 未规划 | v0.1.2 已完成 Vue 3 | 纳入正式规划 |
| **插件数据存储** | 未明确 | 讨论确定三种方案 | 写入插件系统文档 |
| **事件总线** | 计划 NATS | 已实现 NATS + 内存回退 | 更新实际架构 |
| **API 端点** | 计划 32 个 | 实际 19 个 | 更新为实际状态 |

## 已实现功能总览（v0.1.0 ~ v0.1.8）

### 后端 API（19 个端点）

| 模块 | 端点 | 状态 |
|:---|:---|:---|
| 健康检查 | `GET /api/v1/health` | ✅ |
| 用户注册 | `POST /api/v1/auth/register` | ✅ bcrypt 密码 |
| 用户登录 | `POST /api/v1/auth/login` | ✅ JWT Token |
| 当前用户 | `GET /api/v1/auth/me` | ✅ JWT 认证 |
| 用户列表 | `GET /api/v1/users` | ✅ 分页 |
| 用户详情 | `GET /api/v1/users/:id` | ✅ |
| 更新用户 | `PUT /api/v1/users/:id` | ✅ JWT 认证 |
| 帖子列表 | `GET /api/v1/threads` | ✅ 搜索/分页/过滤 |
| 帖子详情 | `GET /api/v1/threads/:id` | ✅ 浏览计数 |
| 创建帖子 | `POST /api/v1/threads` | ✅ JWT 认证 |
| 更新帖子 | `PUT /api/v1/threads/:id` | ✅ 仅作者 |
| 删除帖子 | `DELETE /api/v1/threads/:id` | ✅ 软删除 |
| 版块列表 | `GET /api/v1/categories` | ✅ |
| 版块详情 | `GET /api/v1/categories/:id` | ✅ |
| 创建版块 | `POST /api/v1/categories` | ✅ JWT 认证 |
| 更新版块 | `PUT /api/v1/categories/:id` | ✅ JWT 认证 |
| 回复列表 | `GET /api/v1/threads/:id/posts` | ✅ 分页 |
| 创建回复 | `POST /api/v1/threads/:id/posts` | ✅ JWT 认证 |
| 事件历史 | `GET /api/v1/events` | ✅ 最近 N 条 |

### 前端页面（6 个）

| 页面 | 路由 | 功能 |
|:---|:---|:---|
| 首页 | `/` | 欢迎、统计、最新帖子 |
| 登录 | `/login` | 邮箱+密码登录 |
| 注册 | `/register` | 用户注册 |
| 帖子列表 | `/threads` | 搜索、分页 |
| 帖子详情 | `/threads/:id` | 内容、标签 |
| 发帖 | `/threads/create` | 创建帖子 |
