## v0.3 — Wasm 安全隔离

**目标**：引入 Wasm 运行时，实现轻量级安全插件

| 任务 | 说明 |
|:---|:---|
| **Wasm Runtime** | 基于 Wazero 的 Wasm 沙箱 |
| **Wasm Host Functions** | Host API 的 Wasm 绑定 |
| **对象池管理** | Wasm 实例复用和生命周期 |
| **资源限制** | 内存上限、执行超时 |
| **三层权限模型** | API 级 + 资源级 + 字段级 |
| **插件存储** | SQLite 独立存储 + PostgreSQL 共享存储 |
| **Go SDK** | Wasm + gRPC 双运行时 SDK |
| **CLI 工具** | 插件创建、构建、测试 |

### 验收标准

- [ ] Wasm 插件安全运行，不影响核心系统
- [ ] 三层权限模型正常工作
- [ ] Go SDK 完整可用
- [ ] 至少 2 个 Wasm 示例插件正常工作
