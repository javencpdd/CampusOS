## 模块依赖关系

```
cmd/server/main.go
    └── internal/server/server.go
            ├── pkg/config          ← 配置管理
            ├── pkg/database        ← PostgreSQL 连接池
            ├── pkg/auth            ← JWT + bcrypt
            ├── pkg/eventbus        ← Event Bus (NATS / Memory)
            ├── pkg/middleware       ← HTTP 中间件
            ├── pkg/response        ← 统一响应体
            ├── internal/core/identity/
            │   ├── domain          ← User 实体
            │   ├── repository      ← UserRepository (Memory / PG)
            │   ├── service         ← UserService
            │   └── handler         ← UserHandler
            └── internal/community/
                ├── domain          ← Thread/Post/Category 实体
                ├── repository      ← Repository (Memory / PG)
                ├── service         ← Service
                └── handler         ← Handler
```

## 认证流程

```
注册: POST /auth/register
  → bcrypt 哈希密码
  → 写入 users + accounts 表
  → 发布 user.created 事件

登录: POST /auth/login
  → 从 accounts 表获取密码哈希
  → bcrypt 验证
  → 签发 JWT Access Token (2h) + Refresh Token (30d)

认证请求: GET /auth/me
  → Authorization: Bearer <token>
  → JWT 中间件验证
  → 注入 user_id 到 context
  → Handler 从 context 获取 user_id
```
