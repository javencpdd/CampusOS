# CampusOS v0.1.2-dev 发布说明

> **版本**：v0.1.2-dev
> **发布日期**：2024-06-24
> **上一版本**：v0.1.1-dev
> **状态**：开发中

## 版本概述

v0.1.2-dev 在 v0.1.1-dev（PostgreSQL 迁移脚本）基础上，新增了完整的 **Vue 3 Web UI 前端**，使 CampusOS 从一个纯 API 服务升级为具备可视化界面的社区平台。

## 功能详情

### 前端页面

| 页面 | 路由 | 功能 | 状态 |
|:---|:---|:---|:---|
| 首页 | `/` | 欢迎信息、用户/帖子统计、最新帖子列表 | ✅ |
| 登录 | `/login` | 邮箱+密码登录，自动存储 Token | ✅ |
| 注册 | `/register` | 用户名+昵称+邮箱+密码注册 | ✅ |
| 帖子列表 | `/threads` | 搜索框、帖子表格、分页 | ✅ |
| 帖子详情 | `/threads/:id` | 标题、内容、作者、浏览量、标签 | ✅ |
| 发帖 | `/threads/create` | 标题、内容、版块、标签（需登录） | ✅ |

### 技术实现

| 特性 | 说明 |
|:---|:---|
| **API 代理** | Vite 开发服务器将 `/api` 请求代理到后端 `http://localhost:8080` |
| **自动 Token** | Axios 请求拦截器自动携带 Authorization Header |
| **401 处理** | 响应拦截器检测 401 自动清除 Token 并跳转登录页 |
| **登录态持久化** | Token 和用户信息存储在 localStorage，刷新不丢失 |
| **响应式导航栏** | 根据登录状态显示不同导航项（登录/注册 vs 用户名/发帖/退出） |
| **Element Plus** | 使用 el-card / el-table / el-form / el-button 等组件 |

## 架构变化

```
v0.1.1-dev:
  浏览器 → curl/Postman → Go API Server (port 8080) → 内存存储

v0.1.2-dev:
  浏览器 → Vue 3 SPA (port 3000) → Vite Proxy → Go API Server (port 8080) → 内存存储
                                            │
                                            └── /api/* → localhost:8080
```

## 启动方式

```bash
# 终端 1：启动后端 API
cd ~/bbs/bbs01/CampusOS
go run ./cmd/server/main.go

# 终端 2：启动前端开发服务器
cd ~/bbs/bbs01/CampusOS/web
pnpm dev
```

访问 `http://localhost:3000` 即可看到 CampusOS 社区界面。
