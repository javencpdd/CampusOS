> **发布日期**：2024-06-25

## 版本概述

v0.1.9-dev 是 v0.1 系列的最后一个版本，目标是**补全权限系统、补充缺失 API、完善前端功能**，使系统从"Demo 级"升级为"可用级"。完成后，v0.1 阶段正式收尾，进入 v0.2 插件系统阶段。

## 计划功能

### 1. RBAC 角色系统（P0）

| 任务 | 说明 |
|:---|:---|
| roles 表 | admin / moderator / member / guest 四个内置角色 |
| user_roles 表 | 用户-角色关联，支持按版块分配版主 |
| permissions 表 | 角色-权限关联（resource + action） |
| PermissionService | 权限检查服务（Check 方法） |
| RequirePermission 中间件 | 路由级权限保护 |
| 注册时自动分配 member 角色 | 新用户默认为普通会员 |

### 2. 补全 API 端点（P0）

| 方法 | 端点 | 说明 | 认证 |
|:---|:---|:---|:---|
| `POST` | `/api/v1/auth/logout` | 用户登出（清除 Refresh Token） | ✅ |
| `POST` | `/api/v1/auth/refresh` | 刷新 Access Token | ✅ |
| `POST` | `/api/v1/users/:id/suspend` | 封禁用户 | ✅ admin |
| `POST` | `/api/v1/users/:id/activate` | 解封用户 | ✅ admin |
| `POST` | `/api/v1/threads/:id/pin` | 置顶帖子 | ✅ admin/moderator |
| `POST` | `/api/v1/threads/:id/unpin` | 取消置顶 | ✅ admin/moderator |
| `POST` | `/api/v1/threads/:id/lock` | 锁定帖子 | ✅ admin/moderator |
| `POST` | `/api/v1/threads/:id/unlock` | 解锁帖子 | ✅ admin/moderator |
| `PUT` | `/api/v1/threads/:thread_id/posts/:id` | 编辑回复 | ✅ 作者 |
| `DELETE` | `/api/v1/threads/:thread_id/posts/:id` | 删除回复 | ✅ 作者 |
| `DELETE` | `/api/v1/categories/:id` | 删除版块 | ✅ admin |

补全后 API 端点总数：**30 个**

### 3. 前端功能完善（P1）

| 页面 | 新增功能 |
|:---|:---|
| 帖子详情页 | 回复列表显示、回复表单、置顶/锁定标记 |
| 帖子列表页 | 置顶帖子优先显示、标签筛选 |
| 用户列表页 | 新增页面，管理员可查看 |
| 导航栏 | 显示用户角色标识 |
