# CampusOS v0.1.5-dev 发布说明

> **版本**：v0.1.5-dev
> **发布日期**：2024-06-24
> **上一版本**：v0.1.4-dev
> **状态**：开发中

---

## 版本概述

v0.1.5-dev 新增了**版块（Category）管理**和**帖子回复（Post）系统**，使社区功能从"只能发帖"扩展为"分版块讨论 + 帖子内回复"的完整社区体验。

## 快速测试

```bash
# 启动（内存模式）
cd ~/bbs/bbs01/CampusOS
./campusos-server

# 注册 + 登录
curl -s -X POST http://localhost:8080/api/v1/auth/register \
  -H "Content-Type: application/json" \
  -d '{"username":"jack","nickname":"Jack","email":"jack@example.com","password":"123456"}' | jq .

TOKEN=$(curl -s -X POST http://localhost:8080/api/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email":"jack@example.com","password":"123456"}' | jq -r '.data.access_token')

# 创建版块
curl -s -X POST http://localhost:8080/api/v1/categories \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $TOKEN" \
  -d '{"name":"技术分享","slug":"tech","description":"技术讨论区"}' | jq .

# 发帖
THREAD_ID=$(curl -s -X POST http://localhost:8080/api/v1/threads \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $TOKEN" \
  -d '{"title":"Hello","content":"World","category_id":"cat_001"}' | jq -r '.data.id')

# 回复帖子
curl -s -X POST "http://localhost:8080/api/v1/threads/$THREAD_ID/posts" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $TOKEN" \
  -d '{"content":"这是一条回复"}' | jq .

# 查看回复列表
curl -s "http://localhost:8080/api/v1/threads/$THREAD_ID/posts" | jq .
```
