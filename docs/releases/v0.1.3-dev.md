## 版本概述

v0.1.3-dev 在 v0.1.2-dev（Vue 3 前端）基础上，新增了**数据库连接层、JWT 认证模块、bcrypt 密码工具、认证中间件和 Makefile**，为 PostgreSQL 持久化和真实认证做好了基础设施准备。

## 新增功能详情

### 1. PostgreSQL 连接池 (`pkg/database/database.go`)

```go
pool, err := database.New("postgres://campusos:campusos_dev@localhost:5432/campusos")
```

| 配置项 | 默认值 |
|:---|:---|
| 最大连接数 | 25 |
| 最小连接数 | 5 |
| 连接最大生命周期 | 1 小时 |
| 空闲连接超时 | 30 分钟 |
| 健康检查周期 | 1 分钟 |

### 2. JWT 认证 (`pkg/auth/jwt.go`)

```go
jwtMgr := auth.NewJWTManager(auth.JWTConfig{
    Secret:     "your-secret-key",
    AccessTTL:  2 * time.Hour,
    RefreshTTL: 30 * 24 * time.Hour,
    Issuer:     "campusos",
})

// 签发 Token
accessToken, _ := jwtMgr.GenerateAccessToken(userID, username)
refreshToken, _ := jwtMgr.GenerateRefreshToken(userID)

// 验证 Token
claims, err := jwtMgr.VerifyToken(accessToken)
// claims.UserID, claims.Username
```

### 3. bcrypt 密码工具 (`pkg/auth/password.go`)

```go
// 哈希密码
hash, _ := auth.HashPassword("123456")

// 验证密码
isValid := auth.CheckPassword("123456", hash) // true
```

### 4. JWT 认证中间件 (`pkg/middleware/auth.go`)

```go
// 路由保护
auth := router.Group("/api/v1")
auth.Use(middleware.JWTAuth(jwtMgr))
{
    auth.GET("/auth/me", userHandler.GetMe)
    auth.POST("/threads", threadHandler.CreateThread)
}
```

中间件自动：
- 从 `Authorization: Bearer <token>` 提取 Token
- 验证 Token 签名和有效期
- 将 `user_id` 和 `username` 注入 gin.Context
- 无效 Token 返回 401 错误

### 5. Makefile 命令

| 命令 | 说明 |
|:---|:---|
| `make build` | 编译后端二进制 |
| `make run` | 编译并运行 |
| `make dev` | 热重载开发模式（Air） |
| `make test` | 运行所有测试 |
| `make lint` | 代码检查 |
| `make migrate-up` | 执行数据库迁移 |
| `make migrate-down` | 回滚数据库迁移 |
| `make docker-up` | 启动 Redis + NATS |
| `make web-dev` | 启动前端开发服务器 |
| `make web-build` | 构建前端 |
